# AntDesign
* 流行的开源UI组件库
  material-ui(国外)
  ant-design(国内蚂蚁金服)

# 使用
1. 安装
   cnpm i antd -s
2. 引入样式
   import 'antd/dist/antd.min.css';
3. 使用
   import { WechatOutlined, WeiboOutlined, SearchOutlined } from '@ant-design/icons'
   import { Button, Timeline, DatePicker,Space } from 'antd';
